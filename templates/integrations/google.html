{% if not oauth_configured %}

Falta configurar OAuth en el servidor.

  1. En Google Cloud Console creá un proyecto y habilitá Google Calendar API.
  2. Creá credenciales OAuth 2.0 (tipo aplicación web).
  3. Agregá esta URI de redirección autorizada:
    {{ redirect_uri }}
  4. En el archivo .env completá: 
    GOOGLE_OAUTH_CLIENT_ID=...
GOOGLE_OAUTH_CLIENT_SECRET=...
GOOGLE_OAUTH_REDIRECT_URI={{ redirect_uri }}
APP_BASE_URL=http://127.0.0.1:8000
  5. Reiniciá el servidor y volvé a esta página.
